# 快速开始指南

## 一键编译

```bash
# 进入项目目录
cd /home/you/cursor/weather

# 运行构建脚本
./build.sh
```

## 使用Android Studio

1. 打开Android Studio
2. 选择 `File` -> `Open`
3. 选择项目目录
4. 等待Gradle同步
5. 点击运行按钮 ▶️

## 使用命令行

```bash
# 清理项目
./gradlew clean

# 编译Debug版本
./gradlew assembleDebug

# 安装到设备
./gradlew installDebug
```

## 安装APK

编译完成后，APK文件位于:
- Debug: `app/build/outputs/apk/debug/app-debug.apk`

安装方法:
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

或者直接传输到手机点击安装。

## 功能说明

应用启动后会自动加载:
- 芗城区当前天气和7天预报
- 龙文区当前天气和7天预报

显示信息包括:
- 当前温度
- 天气状况
- 风向和风力
- 湿度
- 7天天气预报详情

## 系统要求

- Android 7.0 (API 24) 或更高版本
- 网络连接
- Android 14 (API 34) 完全兼容

## 故障排除

### 编译失败
1. 检查JDK版本 (需要17+)
2. 检查Android SDK路径
3. 创建 `local.properties` 文件

### 网络请求失败
1. 检查网络连接
2. 确认API Key有效
3. 检查权限配置

### 应用无法安装
1. 确保设备Android版本 >= 7.0
2. 允许安装未知来源应用
3. 检查APK签名
