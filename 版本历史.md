# 版本历史

## v1.0.0 (2024-01-01)

### 首次发布

**新功能**:
- ✓ 显示漳州市芗城区和龙文区的天气信息
- ✓ 实时当前天气（温度、状况、风向、湿度）
- ✓ 7天天气预报
- ✓ Material Design界面设计
- ✓ 支持日夜间模式
- ✓ 网络请求错误处理
- ✓ 数据加载进度提示

**技术特性**:
- 使用Retrofit 2.9.0进行网络请求
- 使用Gson 2.10.1进行JSON解析
- 使用RecyclerView展示列表
- 使用CardView进行卡片布局
- 支持Android 7.0 - Android 14
- compileSdk: 34 (Android 14)
- targetSdk: 34 (Android 14)
- minSdk: 24 (Android 7.0)

**已知问题**:
- 应用图标使用占位图标
- 不支持离线缓存
- 不支持下拉刷新
- 不支持自动定位

**系统要求**:
- Android 7.0或更高版本
- 网络连接
- 至少50MB存储空间

## 更新日志格式说明

### 类型标记
- **新功能**: 全新添加的功能

- **修复**: Bug修复
- **移除**: 删除的功能
- **安全**: 安全性更新

### 优先级
- **高**: 重要功能或严重Bug修复
- **中**: 一般功能或优化
