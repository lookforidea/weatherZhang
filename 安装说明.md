# 安装说明

## 系统要求

- Android 7.0 (API 24) 或更高版本
- 网络连接
- 至少 50MB 存储空间

## 安装方法

### 方法1: 使用ADB安装（推荐）

1. 在电脑上安装 Android SDK Platform Tools
2. 在Android设备上启用USB调试：
   - 进入 `设置` -> `关于手机`
   - 连续点击 `版本号` 7次，启用开发者选项
   - 返回 `设置` -> `开发者选项`
   - 启用 `USB调试`

3. 连接设备到电脑，执行：
```bash
# 检查设备是否连接
adb devices

# 安装APK
adb install app/build/outputs/apk/debug/app-debug.apk

# 或者强制覆盖安装
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

### 方法2: 直接在设备上安装

1. 将APK文件传输到Android设备
   - 使用USB数据线
   - 通过微信/QQ等传输
   - 使用云盘下载

2. 在设备上启用"未知来源"：
   - Android 8.0+: `设置` -> `应用和通知` -> `特殊应用权限` -> `安装未知应用` -> 选择浏览器或文件管理器，允许安装
   - Android 7.x: `设置` -> `安全` -> 启用 `未知来源`

3. 使用文件管理器找到APK文件，点击安装

### 方法3: 通过Android Studio安装

1. 打开Android Studio
2. 连接设备或启动模拟器
3. 点击运行按钮 ▶️
4. 选择目标设备
5. 应用会自动安装并启动

## 权限说明

应用需要以下权限：

### 必需权限
- **网络访问** (`INTERNET`): 获取天气数据
- **网络状态** (`ACCESS_NETWORK_STATE`): 检查网络连接

这些权限会自动授予，无需手动操作。

## 首次运行

1. 启动应用
2. 应用会自动加载芗城区和龙文区的天气数据
3. 如果加载失败，请检查：
   - 网络连接是否正常
   - 是否允许应用访问网络
   - 尝试下拉刷新

## 卸载方法

### 方法1: 在设备上卸载
- 长按应用图标
- 拖动到"卸载"或"删除"
- 确认卸载

### 方法2: 在设置中卸载
- 进入 `设置` -> `应用` 或 `应用管理`
- 找到"漳州天气预报"
- 点击"卸载"

### 方法3: 使用ADB卸载
```bash
adb uninstall com.weather.app
```

## 故障排除

### 问题1: 无法安装
**解决方案**:
- 确保Android版本 >= 7.0
- 检查存储空间是否充足
- 启用"未知来源"安装
- 卸载旧版本后重新安装

### 问题2: 安装后闪退
**解决方案**:
- 检查Android版本兼容性
- 清除应用数据和缓存
- 重启设备后重试
- 重新下载APK文件

### 问题3: 无法获取天气数据
**解决方案**:
- 检查网络连接
- 允许应用使用网络数据
- 检查Wi-Fi或移动数据是否正常
- 尝试切换网络（Wi-Fi ↔ 移动数据）

### 问题4: 应用提示"安全风险"
**解决方案**:
- 这是正常的，因为应用未在应用商店发布
- 应用是安全的，可以选择"仍然安装"
- 源代码完全开放，可以自行检查

## 更新说明

如需更新应用：
1. 下载新版本APK
2. 直接安装（会覆盖旧版本）
3. 数据和设置会保留
