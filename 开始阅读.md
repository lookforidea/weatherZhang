## 🎯 项目简介

这是一个Android天气预报应用，显示福建省漳州市芗城区和龙文区的实时天气和7天预报。

**核心功能**:
- 📍 显示芗城区和龙文区天气
- 🌡️ 实时温度、天气状况、风向、湿度
- 📅 7天天气预报
- 🎨 Material Design界面
- 🌓 支持日夜间主题
- 📱 兼容Android 7.0 - Android 14

## ⚡ 5分钟快速入门

### 如果您想立即编译：
```bash
cd /home/you/cursor/weather
./build.sh
```

### 如果您想用Android Studio：
1. 打开Android Studio
2. File → Open → 选择项目目录
3. 等待Gradle同步
4. 点击运行 ▶️

### 如果您想查看代码：
主要代码在 `app/src/main/java/com/weather/app/` 目录下：
- `MainActivity.java` - 主界面
- `adapter/ForecastAdapter.java` - 列表适配器
- `api/` - 网络请求
- `model/` - 数据模型
