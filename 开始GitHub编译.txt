方法2: 手动命令上传
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

cd /home/you/cursor/weather

# 初始化
git init
git add .
git commit -m "Android Weather App"

# 关联仓库(替换用户名和仓库名)
git remote add origin https://github.com/用户名/仓库名.git

# 推送
git branch -M main
git push -u origin main

方法3: 网页上传 (无需Git)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 打包项目:
   使用已生成的压缩包:
   /home/you/cursor/weather-app-20251223_143216.zip

2. 上传到GitHub:
   a) 在GitHub创建新仓库
   b) 点击 "uploading an existing file"
   c) 上传zip文件
   d) 点击 "Commit changes"

3. 解压到根目录:
   上传后需要将文件移到根目录
   (或直接上传解压后的文件)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
上传后的步骤
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 访问你的GitHub仓库

2. 点击顶部 "Actions" 标签
   
3. 查看 "Build Android APK" 工作流
   - 绿色✓ = 编译成功
   - 黄色● = 正在编译
   - 红色✗ = 编译失败

4. 等待编译完成 (约5-10分钟)

5. 下载APK:
   a) 点击成功的工作流
   b) 滚动到底部 "Artifacts"
   c) 下载 "app-debug-apk"
   d) 解压得到 app-debug.apk

1. 进入 Actions 标签
2. 点击左侧 "Build Android APK"
3. 点击右侧 "Run workflow"
4. 选择分支 (main)
5. 点击绿色 "Run workflow" 按钮
